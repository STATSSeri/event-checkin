本サービスはサブスクリプション型の SaaS サービスであり、デジタルコンテンツの性質上、お客様都合による返金は原則お受けしておりません。

- ユーザーは管理画面からいつでも本サービスを解約できます
- 一旦お支払いいただいた料金は、当社の責に帰すべき事由がない限り返金いたしません
- 月額プランの場合、月の途中で解約した場合でも当該月の利用料金は日割計算されず、全額が発生します
- 年額プランの場合、契約期間中の途中解約による日割返金は行いません
