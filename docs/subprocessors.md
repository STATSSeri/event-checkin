# サブプロセッサー一覧

S/PASS（イベント受付システム）がユーザーデータの処理を委託する第三者事業者の一覧。
プライバシーポリシー第三者提供条項およびセキュリティホワイトペーパーの参照用。

最終更新：2026-05-19

---

## サブプロセッサー

| 事業者 | 役割 | 取扱データ | データ所在地 | 認証・コンプライアンス |
|---|---|---|---|---|
| Supabase Inc.（米国） | 認証基盤・データベース・ファイルストレージ | アカウント情報、イベント・参加者データ、ログ | AWS 東京リージョン（ap-northeast-1） | SOC 2 Type II / HIPAA |
| Vercel Inc.（米国） | アプリケーションホスティング・CDN・WAF | リクエスト／レスポンス（一時処理） | グローバルエッジ（東京近傍を優先） | SOC 2 Type II / ISO 27001 |
| Resend Inc.（米国） | トランザクションメール配信 | 宛先メールアドレス、メール本文 | 米国（AWS） | SOC 2 Type II |
| Stripe Inc.（米国） | 決済処理・サブスクリプション管理 | 決済情報、請求情報 | グローバル（PCI DSS Level 1 環境） | PCI DSS Level 1 / SOC 1, 2 / ISO 27001 |
| Upstash Inc.（米国） | レート制限用 Redis | IP アドレス／ユーザー ID（短期保管） | AWS 東京リージョン | SOC 2 Type II |

---

## 各事業者の役割詳細

### Supabase
- 認証（メールアドレス + パスワード、TOTP 多要素認証）
- PostgreSQL データベース（イベント、参加者、サブスクリプション、パスワード履歴 等）
- ファイルストレージ（現状未使用、将来用に契約のみ）
- データの暗号化保管（AES-256 at-rest）、自動バックアップ

### Vercel
- Next.js アプリケーションのデプロイ・配信
- グローバル CDN による高速配信
- Web Application Firewall（WAF）
- DDoS 緩和

### Resend
- 招待メール、リマインドメール、QRコードメール、お知らせメールの配信
- 配信ステータス（bounce / complaint）の Webhook 受信

### Stripe
- クレジットカード決済の処理
- 月額サブスクリプション・トライアル管理
- 領収書発行・カスタマーポータル提供
- **弊社ではクレジットカード番号を保持せず**、Stripe の Hosted Checkout 経由で処理

### Upstash
- ブルートフォース対策のレート制限（認証 API、メール送信 API 等）
- 識別子（IP アドレスまたはユーザー ID）は短期間（60 秒〜1 時間）のみ保管
